# Language Packs

QuickSend has built-in `zh-CN` and `en-US` languages.

The app chooses the display language in this order:

1. User setting `language`
2. System locale when `language` is `system` or unset
3. `zh-CN` fallback when the locale is not available

Custom language packs are JSON files placed in:

```text
%APPDATA%\quicksend\languages
```

The Settings page shows the exact directory on the current machine.

## File Format

Create a file such as `ja-JP.json`:

```json
{
  "id": "ja-JP",
  "name": "日本語",
  "translations": {
    "nav.phrases": "フレーズ",
    "nav.expansions": "入力展開",
    "nav.clipboard": "クリップボード",
    "nav.apps": "アプリ",
    "nav.settings": "設定",
    "tray.quit": "終了",
    "tray.tooltip": "QuickSend - クイック貼り付け",
    "tray.settingsTitle": "QuickSend 設定"
  }
}
```

Only keys included in the file are overridden. Missing keys fall back to the built-in Chinese text.

Tray menu labels support these optional keys:

- `nav.settings`
- `nav.expansions`
- `tray.quit`
- `tray.tooltip`
- `tray.settingsTitle`

Restart QuickSend after adding or editing a language pack.
