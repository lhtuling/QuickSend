# QuickSend 用户指南

这份文档面向日常使用者，说明如何管理短语、呼出面板、设置文本扩展和备份数据。

## 启动应用

首次启动时会打开设置窗口。之后应用会常驻系统托盘，左键托盘图标可重新打开设置窗口，也可以在托盘菜单中退出应用。

如果开启了开机自启动，QuickSend 会以后台模式启动，不会自动弹出设置窗口。

## 管理短语

在「短语管理」页可以完成这些操作：

- 新建、编辑和删除分组。
- 新建、编辑和删除短语。
- 从剪贴板快速创建短语。
- 为短语设置文本内容、图片内容、缩写和独立热键。
- 通过搜索框按标题、内容、缩写或中文拼音查找短语。

文本短语支持多行内容。图片短语可以通过选择图片文件或在编辑窗口里直接粘贴剪贴板图片创建。

删除分组会同时删除该分组下的短语，操作前请先确认是否已经备份。

## 快捷面板

按 `Ctrl+Alt+Q` 呼出快捷面板。面板会尽量显示在鼠标附近，方便在当前工作位置快速选择短语。

面板内可使用：

| 操作 | 方法 |
| --- | --- |
| 搜索短语 | 直接输入关键词 |
| 选择上一条或下一条 | `ArrowUp` / `ArrowDown` |
| 粘贴短语 | `Enter` 或单击 |
| 只复制短语 | 右键短语 |
| 切换分组 | `Tab` / `Shift+Tab` |
| 关闭面板 | `Esc` |

搜索支持标题、内容、缩写和中文拼音。例如短语标题为「你好」，可以尝试输入 `nh` 或 `nihao`。

## 文本扩展

文本扩展用于把短缩写替换为完整文本。例如把 `;sig` 展开为一段签名。

使用步骤：

1. 在「文本扩展」页新增规则。
2. 填写缩写和展开后的完整文本。
3. 在任意输入框中输入缩写。
4. 按下并松开 `Alt`。
5. QuickSend 会删除缩写并粘贴完整文本。

系统托盘菜单中可以临时切换文本扩展开关。

## 独立短语热键

编辑短语时，可以在「独立热键」字段中按下组合键，例如 `Ctrl+Shift+1`。保存后，无需打开快捷面板，直接按该组合键即可粘贴对应短语。

建议避免占用系统或常用软件已有快捷键。

## 进程规则

进程规则可以让不同应用默认打开不同分组。例如：

| 前台应用 | 默认分组 |
| --- | --- |
| `Code.exe` | 代码片段 |
| `chrome.exe` | 常用回复 |
| `Outlook.exe` | 邮件模板 |

设置方式：

1. 打开「进程规则」页。
2. 新建规则。
3. 输入进程名，或点击获取当前前台应用。
4. 选择默认分组并保存。

当快捷面板打开时，QuickSend 会优先匹配当前前台应用的规则；没有匹配时使用全局默认分组。

## 备份与恢复

在「设置与备份」页可以导出或导入 JSON。

- 导出：生成包含分组、短语、文本扩展和进程规则的 JSON 文件。
- 导入：用 JSON 内容覆盖当前数据。

导入会清空并替换现有数据，建议先导出现有备份。

## 常见问题

### 按快捷键没有反应

先确认 QuickSend 正在运行并显示在系统托盘中。然后检查当前系统或其他软件是否占用了 `Ctrl+Alt+Q`。

### 文本扩展没有触发

确认文本扩展规则已启用，并且托盘菜单没有关闭文本扩展。输入缩写后需要按下并松开 `Alt` 才会触发。

### 图片短语不能粘贴

图片短语依赖系统剪贴板图片能力。请确认目标应用支持粘贴图片，并尝试先右键短语复制，再手动粘贴。

### Linux 上无法获取鼠标位置或前台进程

Linux 实现依赖 `xdotool`。请确认已经安装，并且当前桌面环境允许读取窗口和输入信息。
